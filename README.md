## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

After React.js, I wanted to try making a mobile app. I'm currently learning using JavaScript, and I'm going to make an app with TypeScript.
<br>
This project is Booking themed. After selecting the location, date and number of people, hotels, villas, residences, etc. can be selected from the screen that appears. On this screen, we can also use the filtering feature to select the options according to our wishes. 
<br>
A nice project to learn React Native.
